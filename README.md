# 🚀 TaxBreaker

AI-powered mobile app to simplify Indian income tax for salaried individuals (FY 2024-25).

---

## ✨ What it does

- 📄 Upload tax documents (Form 16, salary slips)
- 🤖 AI analysis using Gemini
- 📊 Tax health score + savings insights
- 💰 Identify missed deductions (80C, 80D, etc.)
- ⚖️ Compare Old vs New Regime
- 🧭 Step-by-step tax strategy
- 🗂️ Track investments (80C tracker)
- 📅 Filing readiness checklist + reminders
- 💬 AI assistant (advisor-style responses)
- 📄 Export clean PDF summary
- 🔒 Privacy-first (data stored locally)

---

## 🧠 Core Idea

TaxBreaker converts complex tax data into:
- Clear insights  
- Actionable steps  
- Better decisions  

No jargon. No confusion.

---

## 🛠 Tech Stack

- React Native (Expo)
- TypeScript
- Gemini AI (1.5 Flash)
- Zustand (state management)
- AsyncStorage (local storage)
- Expo APIs (Document Picker, File System, Notifications)

---

## 📱 Key Features

### 1. Document Analysis
Upload → Extract → AI Analysis → TaxResult

### 2. Strategy Engine
- Personalized action plan  
- Priority-based steps  
- Law references (80C, 80D, 87A)

### 3. Simulator
- Modify inputs  
- Instantly see tax impact  

### 4. AI Assistant
- Context-aware responses  
- Based on your tax data  

### 5. Vault
- Stores documents + analysis  
- Persistent local data  

---

## 🔄 Data Flow
Upload → Extract → Analyze (AI) → TaxResult → Strategy
→ Store → UI → Chat / PDF / Simulator
